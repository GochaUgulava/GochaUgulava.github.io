მთავარზე გადასვლა


როგორ გავიგოთ არის თუ არა სისტემაში დაინსტალირებული Python და/ან მისი კომპონენტები
(Windows ოპერაციული სისტემისათვის)


შევამოწმოთ სისტემაში ინსტალირებულია თუ არა Python

1. კლავიატურაზე ერთდროულად ვაჭერთ WIN + R


2. ავკრიბოთ სტრიქონი cmd შემდეგ მაუსით დავაწკაპუნოთ OK-ზე

3. გამოტანილ ტერმინალის ფანჯარაში ავკრიბოთ ბრძანება python --version კლავიატურაზე დავაწვეთ Enter ღილაკს
მნიშვნელოვანია! პასუხად მივიღოთ Python 3..(რიცხვები), რაც გვიდასტურებს, რომ სისტემაში დაინსტალირებულია მესამე ვერსიის Python (ვერსიის ზუსტ ნომერს სწორედ ეს რიცხვები წარმოადგენს)
თუ ვერ ვიღებთ ამგვარ პასუხს შეეცადეთ აკრიბოთ python3 --version, თუ ამ შემთხვევაშიც ვერ ვიღებთ ვერსიის ნომრის შესახებ შეტყობინებას, მაშინ სისტემაში Python დაინსტალირებული არ არის.


თუ სისტემაში Python დაინსტალირებული არაა, მის დასაინსტალირებლად გადავიდეთ შემდეგ ბმულზე და მივყვეთ ინსტრუქციებს

Python-ის ოფიციალური საიტი

Python-ის არსბობაში რომ დავრწმუნდებით შეგვიძლია შევამოწმოთ დაინსტალირებულია თუ არა მისი სხვადასხვა კომპონენტი.
ეს მეთოდი გამოდგება სხვადასხვა მოდულების შესამოწმებლად, თუმცა ჩვენ მას განვიხილავთ pygame მოდულის მაგალითზე.

1. ტერმინალის ფანჯარაში (პ. 3) შევიყვანოთ ბრძანება python, დავაწვეთ Enter ღილაკს.

2. გამოტანილ მოწვევაში   >>>  შევიყვანოთ ბრძანება: import მოდულის დასახელება
ანუ ამ კონკრეტულ შემთხვევაში import pygame, დავაწვეთ Enter ღილაკს.


თუ სისტემაში ინსტალირებულია ეს მოდული გამოვა წარწერა pygame სხვადასხვა ციფრების თანხლებით, რაც მოდულის დაინსტალირებას ადასტურებს.


ზოგიერთი მოდულისათვის დაინსტალირების შემთხვევაში შეიძლება რაიმე წარწერა არ გამოვიდეს და გამოჩნდეს კვლავ მოწვევა:   >>>


თუ ასე არ მოხდა, ან მოწვევის წინ გამოტანილ იქნა შეტყობინება შეცდომის შესახებ, ესე იგი დასაინსტალირებელი გვაქვს ეს მოდული.


მოდულის დაინსტალირებისათვის გამოვიდეთ python გარემოდან ბრძანებით exit() - ფრჩხილებში არაფერს ვწერთ და ერთმანეთის მიყოლებით ვკრიბავთ ადგილის გამოტოვების გარეშე, ვაწვებით კლავიატურაზე Enter. შემდგომ ოპერაციული სისტემის მოწვევაში (როგორც პ. 3-ში) ავკრიბოთ ბრძანება:
python -m pip install --user მოდულის სახელი
ამ კონკრტულ შემთხვევაში იქნება: python -m pip install --user pygame და დავაწვეთ Enter ღილაკს.


შენიშვნა:  ზოგჯერ ყველგან სადაც ვკრიბავდით ბრძანებას python, შეიძლება ვერ მივიღოთ სასურველი შედეგი. ამ შემთხვევაში ვცადოთ ბრძანება python-ის  ამოცვლა ბრძანებით python3 და ისე გავიმეოროთ ზემოთ მოყვანილი ნაბიჯები.

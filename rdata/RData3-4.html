3.4 ვექტორები

R-ში შევხვდებით მონაცემთა სხვადასხვაგვარ სტრუქტურებს. ყველაზე უმთავრესი და უმარტივესი მათ შორის ვექტორებია. ვექტორი - ერთი და იმავე ტიპის მონაცემთა ერთგანზომილებიანი მასივია. ვექტორის შესაქმნელად გამოიყენება ფუნქცია c(). ფრჩხილეებში ჩამოითვლება ის ელემენტები, რომლებისაგანაც შედგება ვექტორი. მაგალითად:

v1 <- c(2, 5, 74, 0, -3)

v2 <- c(“Grdv”, “one”, “45”, “h”)

v3 <- c(TRUE, FALSE, FALSE)

როგორც ვხედავთ, v1 ცვლადი შეიცავს ვექტორს, რომელიც ხუთი ელემენტისაგან შედგება, ხუთივე ელემენტი რიცხვითი ტიპისაა (ანუ რიხვებია); v2 ცვლადში ოთხი ტექსტური ელემენტისაგან („45“ თითქოს რიცხვია, მაგრამ რადგანაც ბრჭყალებშია მოთავსებული, ის ტექსტად აღიქმება) შემდგარი ვექტორია; v3 ცვლადში კი სამი ლოგიკური ტიპის (მხოლოდ ორი მნიშვნელობის მიმღები -  TRUE ჭეშმარიტი, FALSE მცდარი) ელემენტიანი ვექტორია. საყურადღებოა:  ერთი ვექტორი მხოლოდ ერთი და იმავე ტიპის მონაცემებს უნდა შეიცავდეს (მაგალითად მხოლოდ რიცხვითი, ან მხოლოდ ტექსტური და ა.შ.). მონაცემთა ტიპებზე (მათ შორის სტრიქონებზე, ლოგიკურ ტიპებზე, ფაქტორებზე) საუბარი მომავალში გვექნება.

v5 <- 45

v6 <- “two”         # ტექსტური ცვლადი

v7 <- TRUE        # ლოგიკური ცვლადი

ბოლო სამი მაგალითი სწორია და დასაშვებია. ამ შემთხვევაში ფორმალურად ითვლება რომ ცვლადს ერთ ელემენტიანი ვექტორი (სკალარი) ენიჭება, რისთვისაც ფუნქცია c() არ არის საჭირო და მინიჭება პირდაპირ ხორციელდება.

ვექტორის ცალკეული ელემენტების გამოძახება ინდექსირების მეშვეობითაა შესაძლებელი. ყველა ელემენტი მათი ადგილმდებარეობის მიხედვითაა გადანომრილი(ინდექსირებული) დაწყებული პირველიდან. ელემენტთა გამოსაყვანად საკმარისია მათი ინდექსის მითითება კვადრატულ ფრჩხილებში.

a <- c(3, 67, 5, 122, 0, -3, 5, 99)  # შეიქმნება a ვექტორი 8 ელემენტისაგან

a[3]    # გამოიტანს a ვექტორის მესამე ელემენტს, რომელიც 5-ის ტოლია

a[1]    # გამოიტანს a ვექტორის პირველ ელემენტს, ანუ 3

a[c(1,4,7,8)]    # გამოიტანს a ვექტორის რამოდენიმე ელემენტს, ანუ მათი ინდექსების მიხედვით 1,4,7,8 ელემენტებს, რაც მოგვცემს: 3  122  5  99. რამოდენიმე ინდექსის მითითებისას საყურადღებოა კვადრატულ ფრჩხილებში c() ფუნქციის გამოყენება, რომელშიც ჩამოითვლება საჭირო ინდექსები

a[1,4,7,8]   # შეცდომაა, რამოდენიმე ინდექსის შერჩევის შემთხვევაში კვადრატულ ფრჩხილებში ინდექსების ჩამოსათვლელად უნდა გამოვიყენოთ ფუნქცია  c()

a[2:7]  # გამოიტანს ვექტორის  მეორედან მეშვიდეს ჩათვლით ელემენტებს, ანუ აისახება 67  5  122  0  -3  5.   ორი წერტილი თანმიმდევრობითობის შესაქმნელად გამოიყენება, ამ შემთხვევაში c() ფუნქცია არ არის საჭირო.

ვექტორებში ახალი ელემენტების ჩამატება ან არსებულის ამოშლა იმავე ვექტორის ხელახლა გადაწერით ხორცილდება

q <- c(2, 6, -7, 1, 4, 0, -4)    # შევქმენით ვექტორი

q <- c(q, 74)  # არსებულ ვექტორს ბოლოში დავამატეთ ელემენტი - რიცხვი 74, ფორმალურად არსებულ ცვლადს q -ს მივანიჭეთ ახალი მნიშვნელობა - ძველი ვექტორი q და დამატებითი ელემენტი 74, ამდენად ვექტორი რვა ელემენტს შეიცავს;

q <- c(q[1:3], 56, q[4:8])  # არსებულ ვექტორში პირველი სამი ელემენტის  შემდგომ ჩავსვით ახალი ელემენტი (56) და შემდეგ მისივე დანარჩენი ელემენტები, ფაქტიურად არსებულ ვექტორში მესამე და მეოთხე ელემნტებს შორის ჩაემატა ახალი ელემენტი, ამდენად ვექტორი ცხრა ელემენტს შეიცავს;

q <- q[2:9]  # ამოვშალეთ პირველი ელემენტი, ფორმალურად ხელახლა გადავწერეთ ძველი ვექტორი პირველი ელემენტის გარეშე;

ინდექსი შეიძლება უარყოფითი იყოს, რაც შესაბამისი ელემენტის ამორიცხვას, მის გარეშე ვექტორს გულისხმობს.  წინა მაგალითი შეიძლება შემდეგნაირადაც ჩაიწეროს:

q <- q[-1]  # იგივე ვექტორი პირველი ელემენტის გარეშე

ვექტორის სიგრძის, მასში ელემენტთა რაოდენობის დასადგენად გამოიყენება ფუნქცია length(). ფრჩხილებში მივუთითებთ საინტერესო ვექტორის სახელს, შედეგად მივიღებთ რიცხვს, რომელიც გგვამცნობს თუ რამდენ ელემენტს შეიცავს ვექტორი, ეს რიცხვი ასევე ვექტორის ბოლო ელემენტის ინდექსის მაჩვენებელიცაა.

q <- c(2, 6, -7, 1)   

length(q)    # პასუხად მივიღებთ 4; q ვექტორში ოთხი ელემენტია, ასევე მისი ბოლო ელემნტის (1) ინდექსიც 4 არის.

q[-length(q)]    # ვექტორი ბოლო ელემენტის გარეშე

q[1:(length(q)-1)]  # ვექტორი ბოლო ელემენტის გარეშე; ინდექსირებულია თანმიმდევრობა პირველი ელემენტიდან ბოლოს წინა ელემენტის ჩათვლით; ჯერ გამოითვლება length(q), რომელის ამ შემთხვევაში  4-ის ტოლია, შემდეგ  (4 – 1) რაც 3-ს უდრის, დაბოლოს გამოიტანება q ვექტორის [1:3] ელემენტები, საბოლოოდ მიივიღებთ: 2  6  -7

ვექტორებზე ჩვეულებრივ არის შესაძლებელი არითმეტიკული და ლოგიკური ოპერაციების შესრულება. აღნიშნული ოპერაციები ელემენტების მიხედვით სრულდება, ანუ ერთი ვექტორის პირველი ელემენტი - მეორე ვექტორის პირველ ელემენტთან, მეორე ელემნტი - მეორე ელემენტთან და ა. შ..

a <- c(2, 6, 4, 0)

b <- c(3, 5, 3, -6)

a + b    # ორი ვექტორი შეიკრიბება ელემენტების მიხედვით, ანუ 2+3, 6+5, 4+3, 0+(-6) და მივიღებთ ახალ ვექტორს 5  11  7  -6

a * b  # გამრავლებისას ელემენტები გამრავლდებიან წყვილ-წყვილად, 2*3, 6*5, 4*3, 0*(-6) და მივიღებთ ახალ ვექტორს 6  30  12  0

ანალოგიურად, ელემენტების მიხედვით, წყვილ-წყვილად სრულდება ლოგიკური ოპერაციებიც:

a < b   # ვექტორების შედარება ხდება თითოეული ელემენტის მიხედვით, 2 და 3, 6 და 5 და ასე შემდეგ. შედეგად მივიღებთ ლოგიკურ ვექტორს, რომელიც თითოეული წყვილის შედარების შედეგს შეიცავს - ჭეშმარიტია თუ მცდარია -  TRUE FALSE FALSE FALSE.

როგორც მოყვანილ მაგალითებში ჩანს, ვექტორების სიგრძე, რომლებზეც ოპერაციებს ვახორციელებდით თანაბარი იყო. რა ხდება როდესაც ვექტორებს სხვადასხვა სიგრძე აქვთ? ამ შემთხვევაში უფრო მოკლე ვექტორი გადამუშავდება, ანუ განმეორებით გადაიწერება თანმიმდევრულად ვიდრე მისი ელემენტების რაოდენობა არ გაუტოლდება უფრო დიდი ვექტორის სიგრძეს, რის შემდეგაც ოპერაცია შერსულდება ჩვეულებრივ.

k <- c(3, 7)

a <- (2, 1, 6, 7, 2, 1, 1)

k * a    # პირველი ვექტორი ორ-ელემენტიანია, მეორე კი შვიდ ელემენტს შეიცავს. მათ გასამრავლებლად პროგრამა პირველი ვექტორის ელემენტებს იმდენჯერ გადაწერს ვიდრე ის შვიდელემენტიანი არ გახდება, ანუ (3, 7) გადამუშავდება როგორც  (3, 7, 3, 7, 3, 7, 3). შემდგომ მოხდება უკვე გათანაბრებული ვექტორების ელემენტების მიხედვით გადამრავლება და მივიღებთ ვექტორს - 6  7  18  49  6  7  3

ვექტორები R-ის ძალზე მნიშვნელოვანი, საბაზო მონაცემთა შენახვის,  დამუშავების და წარმოდგენის სტრუქტურაა.  სხვადასხვა ოპერაციებს, ფუნქციებს R-ის პროგრამა ვექტორებზე  (მის შემადგენელ ელემენტებზე)   ერთიანად, როგორც ერთ მთლიან ერთეულზე  ისე ასრულებს.  ეს კი ზრდის პროგრამის  ეფექტურობას და სწრაფქმედებას. ამ მოვლენას ვექტორიზაციას უწოდებენ და მას უფრო დეტალურად მომავალში  გავეცნობით.

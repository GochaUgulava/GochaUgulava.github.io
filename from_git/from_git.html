მთავარზე გადასვლა


როგორ გადმოვწეროთ github-დან და გავუშვათ python თამაშები
(Windows ოპერაციული სისტემისათვის)


GitHub რეპოზიტორიუმიდან Python თამაშის გადმოსაწერად და გასაშვებად შესასრულებელია რიგი მოქმედებები. აქ ამ მოქმედებებს განვიხილავთ Covid Invasion თამაშის მაგალითზე, თუმცა სხა შემთხვევაშიც ქმედებები იდენტურია.

1. ვებ-ბრაუზერით შევითედ შესაბამის GitHub რეპოზიტორიუმში და დავაწკაპუნოთ მწვანე ფერის ღილაკზე Clone or download. ჩამოშლილ მცირე ზომის მენიუში დავაწკაპუნოთ Download ZIP.
ამ პროცედურის გასამარტივებლად, უფრო იოლია, ამ საიტის თამაშის შესაბამის გვერდზე (გადავდივართ მთავარი გვერდიდან) მოვძიოთ პირდაპირი ბმული და მასზე დაწკაპუნებით მივიღებთ შედეგს. ორივე შემთხვევაში საიტიდან კომპიუტერში ჩამოიტვირთება თამაშის არქივირებული ZIP ფაილი


2. ბრაუზერში ფაილის ჩამოტვირთვის დასრულების შემდეგ ჩამოტვირთული ფაილის მენიუში ავირჩიოთ Show in folder, რათა გავხსნათ ის ფოლდერი, რომელშიცაა ჩამოტვირთული ჩვენი ფაილი


3. გახსნილ ფოლდერში მოვნიშნოთ ჩამოტვირთული ფაილი, მის კონტექსტურ მენიუში (რომელიც გამოიტანება ფაილზე მაუსის მარჯვენა ღილაკზე დაწკაპუნებისას) ავირჩიოთ პუნქტი Cut (Вырезать)


4. გადავიდეთ დესკტოპე,  ცარიელ ადგილზე მაუსის მარჯვენა ღილაკზე დაწკაპუნებით გამოტანილ კონტექსტურ მენიუში ავირჩიოთ Paste (Вставить). შედეგად ფაილი გადმოტანილი იქნება დესტოპზე, მითითებულ ადგილზე.


5. მოვნიშნოთ ფაილი, მაუსის მარჯვენა ღილაკზე დაწკაპუნებით გამოტანილ კონტექსტურ მენიუში ავირჩიოთ პუნქტი Extract here (Извлечь в текущую папку). შედეგად არქივი გაიშლება.


6. გაშლილი არქივის ფოლდერს ექნება იგივე სახელი, რაც ფაილს.


7. გავხსნათ ფოლდერი, მასში დავინახავთ თამაშის ფაილებს


8. ვიდრე თამაშს გავუშვებდეთ, დავრწმუნდეთ დაინსტალირებული არის თუ არა სისტემაში Python და სხვა საჭირო მოდულები (თამაშებისათვის ხშირად - მოდული Pygame). შესაბამისი ინსტრუქცია მოცემულია ბმულზე:

როგორ გავიგოთ არის თუ არა სისტემაში დაინსტალირებული Python და/ან მისი კომპონენტები

9. თამაშის გასაშვებათ ორჯერ უნდა დავაწკაპუნოთ თამაშის სახელის მქონე ფაილზე. ამ შემთხვევაში რადგანაც თამაშია Covid Invasion, ფაილის სახელიცაა covid_invasion.py

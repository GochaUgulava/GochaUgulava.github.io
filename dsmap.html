1. Mathematics

Linear Algebra

  • Basic properties of a matrix and vectors: scalar multiplication, linear transformation, transpose, conjugate, rank, and determinant
  • Inner and outer products, matrix multiplication rule and various algorithms, and matrix inverse
  • Matrix factorization concept/LU decomposition, Gaussian/Gauss-Jordan elimination, solving linear system of an equation
  • Eigenvalues, eigenvectors, diagonalization, and singular value decomposition
  • Special matrices: square matrix, identity matrix, triangular matrix, the idea about sparse and dense matrix, unit vectors, symmetric matrix, Hermitian, skew-Hermitian and unitary matrices
  • Vector space, basis, span, orthogonality, orthonormality, and linear least square

 

Single and Multivariate Calculus

  • Functions of single and several variables
  • Derivatives and gradients
  • Step, Sigmoid, Logit, ReLU (Rectified Linear Unit) and Cost functions
  • Plotting of functions,
  • Minimum and Maximum values of a function

 

Probability Theory and Distributions

  • Probability, Multinomial probability, Conditional probability,  Bayes’s theorem
  • Random variables, distributions, expectation, and variance
  • The law of large numbers, the central limit theorem, and the Poisson approximation
  • Infinite sample spaces and axiomatic probability, Measure theory, Probability density functions
  • Conditional expectation and least squares prediction
  • The Poisson process and the Brownian motion process
  • Stochastic processes, Stationary processes, Markovian processes

 

Optimization Methods

  • Cost function/Objective function
  • Likelihood function
  • Error function
  • Gradient Descent Algorithm and its variants

5. Machine Learning

Top

Supervised Learning

Gradient Descent: batch,stochastic; Naive Bayes; Ordinary, Partial Least Squares;  Classification, Regression (Linear, Logistic); Gradient Boosting Regression; Multivariate Adaptive Regression Splines(MARS); ARIMA; Support Vector Machine (SVM); Kernel Smoothing Methods; Kernel Density Estimation; Radial Basis Functions;

Unsupervised Learning

Clustering;  K-means Clustering; Hierarchical Clustering; Dimensionality Reduction:  Principal Component Analysis (PCA),  Linear Discriminant Analysis (LDA),  Kernelized Principal Component Analysis (KPCA); 

Reinforsment Learning

TF agents;  Optimising Rewards; Q-learning; Monte Carlo Methods

Perfomance metrics

AIC, BIC;  Accuracy;  Confusion Matrix;  Presision, Recall, Specifity, F1 score;  AUC-ROC;  Mean Absolute Percentage Error (MAPE);  Root Mean Square Error (RMSE);  Multi co-linearity: Detection (Tolerance, VIF, Correlation Matrix) and Treatment;

Hyperparameter tuning

Overfitting, Underfitting;  Learning Curve, Validation Curve; Bias-Variance Trade-off;  Holdout cross-validation, K-fold cross-validation; Resampling methods; Bootstrapping;  Regularization methods - Ridge, LASSO; Grid-Search, Randomized Search; Curse of Dimensionality; No Free Lunch Theorem; Occams Razor;

Statistical ML

K-Nearest Neighbors (KNN);  Decision Trees;  Classification and regression trees(CARTs);  C4.5, C5.0; Learning Vector Quantization (LVQ);  Iterative Dichotomiser (ID3);  Chi-squared Automatic Interaction Detection (CHAID);

Ensemble models

Random Forests;  Voting Classifiers;  Bagging, Boosting, Stacking, Blending; Dropout; AdaBoost;

Natural Language Processing (NLP)

Word Sense Disambiguation; Pronoun Resolution;  Machine Translation; Tokenization; Regular Expressions; Stemming; Lemmatization;

Neural Networks

Perceptron; Convolutional Neural Networks (CNN); Recurrent Neural Networks (RNN): Sequence-to-sequence, Elman and Jordan Networks, LSTM, GRU, Hopfield Network, Bidirectional, Continuous-time; Generative Adversarial Networks (GAN);  Deep Boltzman Machine; Deep Q-networks (DQN); Recursive Neural Networks; Variative Auto Encoder (VAE); Bayesian Neural Networks; Neural Turing Machine;
